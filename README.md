# OPMVS: OPen-World Multi-View Stereo with Aerial Imagery

Official project repository for **OPMVS: OPen-World Multi-View Stereo with Aerial Imagery**.

> The paper, code, trained models, evaluation scripts, and data preparation instructions will be released here after the publication process permits public distribution.

## Overview

OPMVS studies open-world multi-view stereo reconstruction for aerial imagery. This repository is prepared as the public release entry point for the paper.
